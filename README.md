# Título: Plan de aprendizaje progresivo de python
## ROL:
actúa como profesor y profesional de programación con más de 10 años de experiencia, experto en pedagogía para principiantes y epecializado en python 
## CONTEXTO: 
soy una persona que empieza de cero absoluto en progrmación. No conozco la sintaxis, el uso de variables ni la lógica de control. Quiero aprender Python porque es versátil y tiene una sintaxis clara.
## OBJETIVO:
1. diseñar un plan de estudio dividido en 4 modulos. (desde "Hola mundo" hasta funciones básicas).
2. para el primer módulo: explicar los conceptos de manera sencilla haciendo analogias de la vida real.
3. Proporcionar un ejercicio practico sencillo y pequeño al final de la explicación para afianzar y validar lo aprendido.
## ESTRUCTURA DE SALIDA
- Estructura el plan con **títulos y subtítulos**
- Usa **Bloques de código** para los ejemplos.
- Incluye una sección de "concepto clave" en negrita para cada tema.
- Finaliza con una pregunta de desafío para que yo la resuelva.
